# Ashoka UI
Ashoka Changemaker App

## Prerequisites
* Node (preferably 5.0 or above)
* Disable auto save in your editor so it doesn't trigger hot reloading all the time

## Development Setup

### Setup
Install the npm modules by running the following

`yarn install`
